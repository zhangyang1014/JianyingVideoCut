# GoldenClip — 剪辑美学配置文件 (Editing_Aesthetic.md)

> 此文件可在网页端「配置管理」页面直接编辑。修改后立即生效于下次审计任务。

## 全局参数

```yaml
silence_threshold: 0.5        # 静音判定阈值（秒），超过此值打 <SIL> 标签
min_segment_duration: 1.5     # 最短保留片段时长（秒）
breath_lead_ms: 150           # 每段保留片段开头预留气口（毫秒）
breath_tail_ms: 100           # 每段保留片段结尾预留气口（毫秒）
retake_char_threshold: 5      # 重说识别：相邻句开头相同字符数阈值
style_mode: immersive         # 风格模式: quick_cut | immersive
```

## 语气词词典

以下词语会被打上 `<FIL>` 标签，在 Filler Removal 规则中处理：

```
嗯、啊、哦、呃、那个、然后、就是说、就是、这个、对对对、好好好、
嗯嗯、啊啊、哎、诶、嗯哼、额、哦哦、对吧、是吧、你知道吗、
怎么说呢、说实话、老实说、坦白讲
```

## 规则开关

| 规则 | 说明 | 状态 |
|------|------|------|
| Rule 1: 重说识别 | 相邻句开头 N 字相同则删前句 | ✅ 启用 |
| Rule 2: 残句清理 | 删除不完整/过短片段 | ✅ 启用 |
| Rule 3: 语气词切除 | 移除 `<FIL>` 标记的废话 | ✅ 启用 |
| Rule 4: 词内去重 | 移除 `<STU>` 标记的结巴 | ✅ 启用 |
| Rule 5: 语义去重 | 跨时段相同观点只保留最优 | ✅ 启用 |
| Rule 6: 句内重复 | 删除修正前的半句 | ✅ 启用 |
| Rule 7: 问答闭环 | 访谈场景保证提问与回答成对 | ✅ 启用 |
| Rule 8: 气口对齐 | 剪辑点预留呼吸空间 | ✅ 启用 |

> 若要临时关闭某条规则，将 `✅ 启用` 改为 `❌ 禁用` 并保存。
